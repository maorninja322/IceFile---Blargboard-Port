# IceFile - Blargboard-Port

## What is it?
This is a port of IceFile as a standalone plugin for blargboard, -plus, -xd.

## Features?
Public uploading

Public - User can upload files and the files will be listed

## How to install:

1. Enable The Plugin
2. Enjoy!
